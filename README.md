## Guides

Educational content covering web3 concepts, EVM, Solana, and security.

## Templates

Starter project scaffolds used by the `w3-kit` CLI.

## Used by

- **w3-kit CLI** — fetches recipes and templates at runtime via GitHub API
- **w3-kit website** — fetches recipes and guides at build time for docs.w3-kit.com
